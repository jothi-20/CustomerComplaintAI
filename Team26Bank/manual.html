End User Manual

1. Login to System

  • Enter valid username and password.
  • Click Login to access dashboard.
  • Invalid credentials will display error message.

2. Create New Case

  • Click "New Case".
  • Enter Customer Reference ID.
  • Select Interaction Type.
  • Select Channel.
  • Select Priority.
  • Enter Raw Notes.
  • Click Submit to create case.

3. AI Case Note Generation

  • Click "Generate AI Note".
  • System converts raw notes into professional format.
  • Interaction Summary
  • Action Taken
  • Next Steps

4. Review & Edit

  • User can edit AI generated content.
  • Make necessary corrections if required.
  • Save changes.

5. Status Workflow

  • Draft
  • Reviewed
  • Finalized

6. SLA Tracking

  • SLA date auto-calculated based on Priority.
  • System tracks SLA breach.
  • Dashboard shows deadlines.

7. Follow-up Management

  • Add follow-up remarks.
  • Mark follow-up as completed.

8. Dashboard Overview

  • Total Active Cases.
  • Cases by Status.
  • Cases by Priority.
  • SLA Compliance Rate.

9. Search & Filter

  • Search by Case ID.
  • Filter by Status, Priority, Channel.
  • View SLA Breached cases.

10. Export & Audit

  • Export finalized notes as PDF or text.
  • Raw Notes
  • AI Generated Notes
  • Final Content
  • Status Changes
  • Timestamps
